Este proyecto corresponde al proceso inicial de maqueta o modelo inicial sobre el cual decidir el contenido de la pagina en desarrollo.

Se ha utilizado la plantilla que se indica más abajo.

Se encuentran algunos inconvenientes para implementar el desarrollo, principalmente es la primera vez en que utilizo un plantilla para acelerar el proceso de diseño. (Se utilizo esta plantilla ya que me agrada la forma en como se encuentran definidas cada una de las secciones de la pagina, aunque se reconoce que la navegación vertical podria ser muy extensa hacia abajo (Cosa para resolver)

Uno de los inconvenientes en la implementación es que los textos tienen asignados estilos de css de forma segmentada y no por el orden de cada sección en la pagina.

Ejemplo : <!-- Banner -->
							<div id="banner">
								<h2 class="texto1-gris"><strong>CargoClick:</strong> Tú mejor opción de entregas en la Región Metropolitana </h2>
								
								<h3 class="texto2-gris">Creada pensando en emprendimientos que comienzan sus ventas - Sí estas facturando o no - Solucionamos tus entregas</a></h3>
								<p class="texto3-gris">Amplia cobertura en Santiago, precios accesibles y mucho más</p>
								
							</div>


Cuenta con 3 textos un h2, h3 y p pero en la hoja de estilos no se encuentran asignados a cada uno de ellos, ya que los selectores se encuentran agrupados y mezclados en distintos puntos del css

Estos textos necesito agruparlos para poder aplicar un selector que asigne valores de estilo a cada etiqueta o clase y asi poder resaltarlos de la imagen de fondo.

Como explico estoy comenzando a entender la aplicación de estilos y se me complico el ejercicio con el uso de la plantilla.
